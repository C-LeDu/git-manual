git commit --amend

La commande git commit --amend permet de faire des modifications sur le dernier commit si vous avez fait des erreurs dans le commentaire ou si vous avez oublié de add un ficher. ATENTION: Cela change l'histoire (et le SHA) donc n'utilisez pas cette dernière si vous avez déja publié vos commits!

Syntaxe

git commit --amend
